# C Version

## Purpose of the C Version

This version demonstrates procedural programming in C by solving the statistics problem with separate functions, explicit loops, array handling, and step-by-step control flow.

## Why It Is Procedural

- The program is organized as procedures that operate on data passed into functions.
- It uses direct array manipulation and explicit size tracking.
- It uses loops to process values step by step.
- Memory and data handling are visible because the program copies and sorts arrays directly.
