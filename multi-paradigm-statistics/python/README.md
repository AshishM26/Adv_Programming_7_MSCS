# Python Version

## Purpose of the Python Version

This version demonstrates object-oriented programming in Python by using a class to store the dataset and provide methods for mean, median, mode, and output formatting.

## Why It Is Object-Oriented

- The program is built around a `StatisticsCalculator` class.
- The dataset is stored as an instance attribute inside the object.
- The methods `mean`, `median`, `mode`, and `display_results` define behavior that belongs to that object.
- The class encapsulates both state and related operations in one clear unit.

## Class Overview

- `__init__` stores the dataset in the object.
- `mean` calculates the average.
- `median` sorts the data and finds the middle value.
- `mode` returns a list of the most frequent value or values.
- `display_results` prints the shared screenshot-friendly output format.
